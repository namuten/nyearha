# 공통수학1 출제 포인트 정리 (킬러/함정 분석)

## 1. 단원별 주력 출제 타사겟

- **[다항식의 연산]**: 곱셈 공식의 변형(세제곱끼리의 합, 세 문자 제곱합)을 이용해 문자의 합/곱/교차 요소를 던져주고 값을 추론하게 하는 연산 문제가 1번~3번 기본기로 나옴.
- **[나머지 정리]**: 다항식 P(x)를 '2차식'이나 '3차식'으로 나누었을 때 나머지를 구하는 문제 극빈출. 이때 R(x)를 상수 r이 아니라 각각 'ax+b' 혹은 'ax²+bx+c'로 먼저 가정하고 식을 세워 수치대입을 해야 풀린다.
- **[복소수 (프린트 연계)]**: 학교 프린트물로 특별 언급된 "음수의 제곱근 곱과 나눗셈 기법". 분수 형태 $\frac{\sqrt{a}}{\sqrt{b}} = -\sqrt{\frac{a}{b}}$ 가 성립한다고 지문에 나오면, 자동반사로 "분자 루트속 a는 양수(>0), 분모 루트속 b는 음수(<0)"라는 조건을 끌어내어 절댓값을 벗기는 킬러 출제가 예측됨.
- **[이차 함수와 직선]**: 포물선 위를 지나가는 직선(y=mx+n)이 접한다는 조건이 주어지면 두 식을 빼서 연립한 새로운 식의 판별식 $D=0$ 을 때려 미지수 k를 찾는 문제 무조건 출제.
- **[이차함수 제한범위 최대최소]**: 단순 꼭짓점이 아니라 공통 2차식을 대문자 T로 **치환**해서 푸는 문제. 치환된 문자 T의 범위가 꼭짓점 기준으로 새로 탄생한다는 점을 놓치면 바로 오답이 된다.

## 2. 절대 피해야 할 함정 포인트 (학생들 가장 많이 틀리는 유형)

> [!CAUTION]
> 학교 선생님이 파놓는 수학 시험 단골 함정들

- **🚨 함정 1: 최고차항 계수 조립제법 오류**
  - **오답 이유**: 다항식을 $(2x-1)$로 나눌 때 조립제법 칸에 $1/2$을 넣고 진행하여 나온 몫의 숫자 배열을 '그대로' 몫이라고 써버림.
  - **정상 풀이**: 1/2을 넣어 나온 결과는 사실 $(x-\frac{1}{2})$로 나눈 결과이므로 나온 몫의 모든 수를 전부 반(1/2)으로 한 번 더 나눠 주어야 제대로 된 몫 $Q(x)$가 된다. 나머지는 변하지 않음.
- **🚨 함정 2: 허수 $i$ 의 덧셈과 곱셈 헷갈림**
  - **오답 이유**: 프린트 범위 - $\sqrt{-2} \times \sqrt{-3}$ 을 계산할 때 루트 안의 $-2$와 $-3$을 무작정 곱해 $\sqrt{6}$이라고 써버림.
  - **정상 풀이**: 실수 체계 법칙이 뚫린다. 각각 $\sqrt{2}i \times \sqrt{3}i$ 로 변환한 뒤 $i^2 = -1$ 임을 곱해 $-\sqrt{6}$ 으로 써야 한다. (부호가 마이너스로 역전되어 튀어나오는지 확인!) 
- **🚨 함정 3: 나머지 차수 망각**
  - **오답 이유**: 다항식을 3차식으로 나누는 검산식 세팅 시 미지의 나머지 $R(x)$를 $ax+b$ 라고만 단순 설정함.
  - **정상 풀이**: 나눈 식(3차)보다 차수가 딱 한 단계 얕은 **최대치**인 $ax^2+bx+c$ 구조로 (2차 이하식) 무조건 세팅하고 시작해야 답이 나온다.
- **🚨 함정 4: 제한 구간 꼭짓점 대입 함정**
  - **오답 이유**: $1 \le x \le 4$ 구간에서 $y = (x-5)^2 + 3$의 최솟값을 구하라고 했는데 아무 생각 없이 완전제곱 괄호가 0이 되는 꼭짓점 Y값 '3'을 답으로 체크함.
  - **정상 풀이**: 꼭짓점의 x좌표는 5이다. 그런데 주어진 조건 범위는 1~4 구역이다! 꼭짓점이 허락된 방어 범위 밖이므로 그림을 그리면 함수는 1부터 4까지 우하향 하강한다. 따라서 최솟값은 무조건 $x=4$일 때이며 '3'이 아니다! (최대 고득점 분수령)
